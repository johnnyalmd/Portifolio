NASA API - Django + JavaScript + API + Chart.js

Este projeto é uma aplicação web interativa desenvolvida com Django, JavaScript e Chart.js, que utiliza várias APIs da NASA para fornecer informações sobre o espaço. Os usuários podem pesquisar imagens da missão Rover com base na data, proporcionando uma visão única das paisagens marcianas. Além disso, o projeto inclui uma funcionalidade que permite aos usuários pesquisar os asteroides mais próximos da Terra, retornando o tamanho e o nome do asteroide em um gráfico visualmente atraente criado com Chart.js. A aplicação também integra a API da Imagem do Dia da NASA, oferecendo aos usuários uma nova e fascinante imagem espacial todos os dias. Este projeto é uma combinação perfeita de ciência e tecnologia, proporcionando uma experiência educativa e envolvente para os usuários
